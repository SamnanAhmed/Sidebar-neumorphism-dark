# Neumorphic Sidebar 🌑✨

A responsive, dark-themed sidebar navigation built using **Neumorphism (Soft UI)** design principles. Features smooth hover glow effects, a collapsible mobile menu, and an expand/reduce toggle for desktop screens.

## 🔥 Features

- **Neumorphism Design** — soft inset shadows giving a realistic pressed/embossed look
- **Glow Hover Effect** — gradient glow animation on link hover
- **Responsive Layout**
  - Mobile/Tablet: slide-in sidebar with hamburger toggle
  - Desktop (≥1150px): fixed sidebar, always visible
- **Expand/Collapse Toggle** — on large screens, shrink sidebar to icon-only view with tooltip labels on hover
- **Clean Vanilla JS** — no frameworks, just plain JavaScript for show/hide and expand logic
- **Remix Icons** — lightweight icon library for menu items

## 🛠️ Built With

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript
- [Remix Icon](https://remixicon.com/)
- Google Fonts (Montserrat)
